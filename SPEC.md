# 朋友聚会网站 - 项目规格说明书

## 项目概述
- **项目名称**: 阳光聚会圈
- **类型**: 社交聚会平台
- **核心功能**: 朋友聚会管理、照片视频分享、能力展示
- **目标用户**: 朋友圈成员

## 功能模块

### 1. 用户系统
- 注册（用户名、密码、头像）
- 登录/登出
- 个人资料编辑

### 2. 好友展示
- 成员列表展示
- 个人名片（头像 + 基本信息）
- 技能介绍
- 能力数值：
  - 酒量（1-10级）
  - 体重（kg）
  - 能喝几场（数字）
  - 能喝几小时（数字）
  - 能连续几天作战（数字）
  - 受欢迎度（1-10级）

### 3. 内容管理
- 照片/视频上传
- 上传需管理员审核（待审核状态）
- 审核通过后公开展示

### 4. 聚会预约
- 发布聚会活动
- 预约参加
- 聚会时间、地点、内容

### 5. 审核后台
- 管理员登录
- 审核上传内容（通过/拒绝）

## 视觉设计
- **风格**: 简约现代 + 阳光暖色系
- **主色调**: 
  - 背景: 暖白色 #FFF8F0
  - 主色: 阳光橙 #FF9500
  - 辅色: 温暖黄 #FFD60A
  - 文字: 深棕 #3D2914
- **字体**: 思源黑体 / Noto Sans SC
- **圆角**: 16px
- **阴影**: 柔和投影

## 页面结构
1. 首页（聚会活动展示）
2. 成员列表（好友展示）
3. 个人主页
4. 上传页面
5. 管理后台
6. 登录/注册

## 技术方案
- 前端: HTML5 + CSS3 + JavaScript
- 后端: Python Flask
- 数据库: SQLite
- 部署: Railway / Render（免费公网）
